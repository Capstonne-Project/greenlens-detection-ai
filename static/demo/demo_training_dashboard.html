📋 Workflow: Tải dataset về → Đưa vào train

          

            

              
Bước 1 — Tải dataset

              

                Tải ZIP từ Roboflow Universe (format YOLOv8).

                Tìm: smoke detection, garbage detection, water pollution
              

            

            

              
Bước 2 — Convert class name

              

                Dùng Remap class names bên dưới.

                Map tên class dataset → TRASH / WATER / SMOKE.

                Download ZIP đã convert.
              

            

            

              
Bước 3 — Restructure (nếu cần)

              

                Nếu ZIP chưa có đúng cấu trúc images/train + labels/train →

                Dùng Restructure để tự động tách train/val.
              

            

            

              
Bước 4 — Upload & Train

              

                Vào tab 🎯 YOLO — Pollution → Upload ZIP → Start training.

                Batch 8 · Epochs 50 · Workers 0 cho máy yếu.
              

            

          

          

            💡 Tip: Nếu dataset Roboflow đã đúng format YOLOv8 và chỉ cần đổi tên class → bỏ qua Bước 3, upload thẳng sau Bước 2.
          

        


        
        

          

          
🗑️ Lọc class — tự động xóa ảnh & label không cần

          

            Upload ZIP dataset. Server đọc data.yaml để liệt kê các class.
            Tick vào class muốn giữ lại — ảnh nào chỉ chứa class bị bỏ sẽ bị
            xóa hoàn toàn (cả ảnh lẫn .txt). YAML được viết lại với class mới, ID được re-index từ 0.

Restructure — đổi cấu trúc thư mục sang chuẩn YOLO

          

            Input ZIP có cấu trúc train/<ảnh> + labels/<txt>
            → Output ZIP chuẩn images/train, images/val,
            labels/train, labels/val sẵn để upload train.

            Label tự convert OBB → AABB nếu cần. Val split mặc định 15%.
          

          

            
            
          

          
          

        


        

          

          
1 Remap class names

          

            Upload dataset ZIP bất kỳ từ Roboflow, Kaggle, v.v. Server đọc data.yaml
            và liệt kê class có trong dataset. Bạn map từng class sang TRASH / WATER / SMOKE
            hoặc SKIP (xóa bỏ). Class ID sẽ được tự động re-index về đúng 0·1·2.
          

          
          
          

        


        

          

          
2 Map class về đúng label dự án

          

            Chọn mỗi class trong dataset tương ứng với class nào của GreenLens (TRASH · WATER · SMOKE).
            Class chọn SKIP sẽ bị xóa khỏi tất cả label — dòng bbox đó không được train.
            Server tự re-index class ID về 0·1·2 và ghi lại data.yaml mới.
          

          

          
          

        


        

          

          
Hướng dẫn

          

            Ví dụ thực tế — Roboflow "Garbage Detection":

            Dataset có: 0: Plastic Trash · 1: bottle · 2: food packaging · 3: person

            → Map 0,1,2 về TRASH · Map 3: person → SKIP

            → Server đổi tất cả label về class_id=0, xóa dòng person, ghi lại data.yaml với nc:1 · 0:TRASH

            → Download ZIP mới → Upload vào tab 🎯 YOLO — Pollution để train.


            Tip quan trọng: Mỗi class từ Roboflow nên được map riêng — đừng merge WATER lẫn SMOKE vào TRASH vì model sẽ không phân biệt được.

            Tải từng dataset riêng biệt cho từng class (TRASH riêng, WATER riêng, SMOKE riêng) → convert từng cái → upload lần lượt vào cùng 1 training job.
          

        

      


      
      

        

          

            
✂️ Tách Dataset

            
Tách ZIP dataset thành 2 phần — ảnh & label khớp tên nhau, cả 2 phần đều được nén và tải về

          

        


        

          

          
Cấu hình tách

          

            Upload file ZIP chứa dataset YOLO (ảnh + label). Nhập số lượng ảnh muốn tách ra.
            Server sẽ ghép đúng cặp image ↔ label theo tên file, rồi trả về 1 ZIP bọc ngoài
            chứa 2 file ZIP bên trong: phần tách và phần còn lại.
          


          

            
            
          


          
          

            
          


          
          

        


        

          

          
Hướng dẫn

          

            Dataset có 1000 ảnh, muốn tách 200 để test:

            → Upload ZIP, nhập 200 → tải về dataset_split_bundle.zip

            → Giải nén ra được 2 file:

              • dataset_split_200.zip — 200 ảnh + label tương ứng

              • dataset_remainder_800.zip — 800 ảnh + label còn lại


            Lưu ý: Ảnh và label được ghép theo tên file (stem).
            Ảnh không có label tương ứng vẫn được tách nhưng sẽ không có file .txt đi kèm.
            File data.yaml và các file chung khác có trong cả 2 ZIP.
