# `docs/plans/` — Kế hoạch triển khai chi tiết

Thư mục này chứa **bản kế hoạch mở rộng** cho từng phase hoặc spike, bám trên `docs/AI_Service_Development_Plan.md`.

- **Khi nào tạo:** Dùng slash command **`/plan`** trong Cursor Agent (workspace command trong `.cursor/commands/`, định nghĩa [Agent Skills — migrate / commands](https://cursor.com/docs/skills.md)) hoặc gọi skill `@greenlens-plan-detail`.
- **Đặt tên gợi ý:** `phase-{N}-{topic-kebab}.md` hoặc `YYYY-MM-DD-{topic-kebab}.md`.
- **Nguyên tắc:** Không thay plan gốc; chỉ bổ sung chi tiết cách làm và acceptance criteria cục bộ.
